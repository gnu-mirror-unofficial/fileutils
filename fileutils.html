Fileutils

The GNU File Utilities are the basic file-manipulation utilities of the GNU operating system.

The tools supplied with this package are:

  • chgrp - Changes file group ownership.
  • chown - Changes file ownership.
  • chmod - Changes file permissions.
  • cp - Copies files.
  • dd - Copies and converts a file.
  • df - Shows disk free space on filesystems.
  • dir - Gives a brief directory listing.
  • dircolors - Setup program for the color output of GNU ls.
  • du - Shows disk usage on filesystems.
  • install - Copies file and sets its permissions.
  • ln - Creates file links.
  • ls - Lists directory contents.
  • mkdir - Creates directories.
  • mkfifo - Creates FIFOs (named pipes).
  • mknod - Creates special files.
  • mv - Moves files.
  • rm - Removes (deletes) files.
  • rmdir - Removes empty directories.
  • shred - Destroy data in files.
  • sync - Synchronizes filesystem buffers and disk.
  • touch - Changes file timestamps.
  • vdir - Long directory listing.

Online Manual

Online Manual for these utilities is available at http://www.gnu.org/software/fileutils/doc/manual/html/fileutils.html.

The manual is available in the info system of the GNU Operating System. Use info to access the top level info page.

Frequently Asked Questions

Please check the FAQ for the core utilities for answers to your most frequently asked questions. This FAQ contains answers to questions asked on the mailing lists and newsgroups.

Downloading

Released versions of GNU fileutils can be found in the subdirectory /gnu/fileutils/ on your favorite GNU mirror. For other ways to obtain fileutils, please read How to get GNU Software

Testing versions of alpha releases can be found on the alpha release server. A different server is used for alpha releases so that people do not confuse testing snapshots and released versions. Alpha test versions are available at ftp://alpha.gnu.org/gnu/fileutils Patches are most appreciated in unified diff format against the latest available testing version.

The latest testing version is ftp://alpha.gnu.org/gnu/fileutils/fileutils-4.1.5.tar.gz

How to Get Help

For general discussion of bugs in the utilities the news group gnu.utils.bug is probably the most appropriate forum. An archive of the bug report mailing list is available at http://mail.gnu.org/pipermail/bug-fileutils/.

How to Help

To contact the maintainers, either to report a bug or to contribute fixes or improvements, send mail to bug-fileutils@gnu.org.
